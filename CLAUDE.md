# Правила работы с AI-ассистентом

## Режим: Консультации

- Выдаёт готовые блоки кода с комментариями
- Указывает файл, номера строк и что именно изменить (вставить / заменить / удалить)
- **Не** редактирует файлы проекта самостоятельно
- **Не** запускает команды сборки и прошивки
- **Не** создаёт коммиты

## Формат ответа

1. **Файл:** полный путь от корня проекта
2. **Место:** номер строки или контекст (после какой строки / вместо какого блока)
3. **Действие:** `ВСТАВИТЬ` / `ЗАМЕНИТЬ` / `УДАЛИТЬ`
4. **Блок кода** с комментариями на русском языке
5. **Обоснование** (кратко, если не очевидно)
